# CNN-Based Crop Disease Detection on FPGA

## Overview

This project implements a lightweight Convolutional Neural Network (CNN) on FPGA to classify crop leaf images as **Healthy** or **Diseased**. The design focuses on low-latency edge AI inference using Verilog HDL and Xilinx Vivado.

---

## Features

- Lightweight CNN architecture
- Crop disease classification
- FPGA implementation
- Low-latency inference
- Edge AI application

---

## Technologies Used

- Verilog HDL
- Xilinx Vivado
- FPGA
- CNN
- Computer Vision

---

## Project Files

- `cnn_top.v` – Top-level CNN module
- `conv3x3.v` – Convolution layer
- `relu.v` – ReLU activation
- `tb_top.v` – Testbench
- `minimal.xdc` – FPGA constraints
